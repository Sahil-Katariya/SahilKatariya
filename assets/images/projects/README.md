# Project Images

This directory contains images for portfolio projects. Replace these placeholder URLs with actual project screenshots.

## Image Guidelines

### Dimensions
- **Project Cards**: 400x250px (16:10 aspect ratio)
- **Project Details**: 800x500px (16:10 aspect ratio)
- **Thumbnails**: 150x94px (16:10 aspect ratio)

### Format
- **Primary**: WebP for modern browsers
- **Fallback**: JPEG for compatibility
- **Quality**: 80-85% for optimal balance

### Naming Convention
- Use kebab-case: `project-name.jpg`
- Include size suffix: `project-name-large.jpg`
- Include format: `project-name.webp`
