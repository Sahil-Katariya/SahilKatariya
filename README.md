# Sahil Katariya - Portfolio Website

A modern, responsive portfolio website showcasing the work and skills of Sahil Katariya, a Full Stack Developer and AI Engineer from Bhavnagar, Gujarat, India.

## 🌟 Features

- **Modern Design**: Clean, professional design with dark/light theme support
- **Responsive Layout**: Optimized for all device sizes from mobile to desktop
- **Interactive Animations**: Smooth scroll animations, particle effects, and micro-interactions
- **Performance Optimized**: Fast loading with lazy loading, image optimization, and efficient code
- **SEO Friendly**: Comprehensive meta tags, structured data, and semantic HTML
- **Accessible**: WCAG compliant with keyboard navigation and screen reader support
- **PWA Ready**: Progressive Web App capabilities with service worker and manifest
- **Modular Architecture**: Clean, maintainable code structure with separated concerns

## 🚀 Quick Start

### Prerequisites
- Node.js 14+ (for build system and development server)
- Modern web browser with ES6+ support

### Development Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/sahilkatariya/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:3000`

### Production Build

1. **Build optimized files**
   ```bash
   npm run build
   ```

2. **Serve production build**
   ```bash
   npm run serve:dist
   ```

## 🛠️ Technology Stack

- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Modern CSS with custom properties, Grid, and Flexbox
- **JavaScript (ES6+)**: Vanilla JavaScript with modern APIs

## 🎨 Customization

All content and settings can be customized through JSON configuration files in the `config/` directory:

- **Site Information**: `config/site.json`
- **Skills & Expertise**: `config/skills.json`
- **Projects Portfolio**: `config/projects.json`
- **Theme & Styling**: `config/theme.json`
